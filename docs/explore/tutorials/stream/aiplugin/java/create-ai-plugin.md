## 创建插件动作
![img.png](/img/explore/stream/aiplugin/create-plugin-action.jpg)

:::info
动作名称 : AI友好的动作名称。最多 20 个字符。<br />
动作唯一标识Key : 用于业务路由。<br />
动作描述 : AI友好的动作描述。（通过关键字）帮助大模型更好的理解并定位动作。最多 500 个字符。<br />
:::
