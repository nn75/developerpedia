# 1. 介绍

本文以及接下来的步骤指引，介绍如何创建并使用 AI 插件。

## 功能

通过这个教程，你可以学会
1. 创建 AI 插件
2. 搭建 AI 插件服务端；
3. 使用 AI 插件搭建 AI 技能；
4. 构建 AI 插件返回数据；
5. 通过数字人/魔法棒触发AI技能
6. (可选)进阶，通过AI技能触发用户代理凭证授权流程并获得用户代理凭证；
